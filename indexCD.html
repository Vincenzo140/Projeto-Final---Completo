Ciência de Dados

Matemática e estatística descritiva

Amostragem

Amostragem é o processo de seleção de uma parte representativa de uma população maior para análise. Em vez de estudar toda a população, uma amostra é coletada e analisada para obter informações sobre o todo. A amostra deve ser escolhida de forma aleatória e representativa, de modo a minimizar vieses e garantir que os resultados sejam generalizáveis para a população. A amostragem é amplamente utilizada em pesquisas de opinião, estudos de mercado, pesquisas científicas e em muitos outros campos, permitindo obter insights e tomar decisões com base em uma fração da população completa.

Aleatória

A amostragem aleatória é um procedimento estatístico que envolve a seleção de uma amostra de forma aleatória e imparcial a partir de uma população. Esse método é baseado no princípio de que cada elemento da população tem a mesma probabilidade de ser escolhido para compor a amostra. Isso garante que a amostra seja representativa e que os resultados obtidos possam ser generalizados para toda a população. A amostragem aleatória é utilizada em várias áreas, como pesquisas de opinião, estudos de mercado, análises estatísticas e muitos outros, sendo uma ferramenta fundamental na obtenção de informações precisas e confiáveis.

Sistemática

A amostragem sistemática é um método de seleção de amostras em que os elementos são escolhidos de forma sistemática, seguindo um padrão predefinido. É um processo eficiente e organizado, no qual todos os elementos têm uma chance igual de serem selecionados. Esse método é amplamente utilizado em pesquisas de opinião, estudos de mercado e em diversas áreas da pesquisa científica.

Estratificada

A amostragem estratificada é um método de seleção de amostras em que a população é dividida em subgrupos ou estratos homogêneos com base em características relevantes. Em seguida, são selecionadas amostras de cada estrato de forma independente, levando em consideração a proporção de cada estrato na população. Esse método permite obter uma representação mais precisa da população e é amplamente utilizado em pesquisas de opinião, estudos demográficos e pesquisas de mercado.

Tendência central

Mádia

A média é uma medida de tendência central que representa o valor médio de um conjunto de dados. É calculada somando todos os valores e dividindo pela quantidade de elementos. A média é amplamente utilizada para representar o valor central de um conjunto de dados e fornece uma visão geral do comportamento dos dados. É uma medida sensível a valores extremos, pois considera todos os valores igualmente. A média é frequentemente usada em áreas como estatística, análise de dados e ciências sociais para resumir e comparar conjuntos de dados.

Mediana

A mediana é uma medida de tendência central que representa o valor do meio em um conjunto de dados ordenados. Para calcular a mediana, os dados devem ser organizados em ordem crescente ou decrescente. Se houver um número ímpar de elementos, a mediana é o valor do meio. Se houver um número par de elementos, a mediana é a média dos dois valores do meio. A mediana é uma medida robusta, menos sensível a valores extremos, e é frequentemente usada quando há a presença de outliers nos dados. É amplamente utilizada em estatística, análise de dados e em situações em que é importante identificar um valor central representativo.

Moda

A moda é uma medida de tendência central que representa o valor mais frequente em um conjunto de dados. É o valor que ocorre com maior frequência. A moda é especialmente útil em conjuntos de dados discretos, onde é possível identificar claramente qual valor se repete mais vezes. Em conjuntos de dados contínuos, pode haver mais de uma moda ou até mesmo nenhum valor que se repita com maior frequência. A moda é uma medida simples de identificar o valor mais comum em um conjunto de dados.

Percentil

O percentil é uma medida estatística que divide um conjunto de dados em 100 partes iguais. Ele é utilizado para identificar a posição relativa de um valor dentro de um conjunto de dados. Por exemplo, o percentil 50, também conhecido como mediana, divide os dados em duas partes iguais, onde 50% dos valores estão abaixo e 50% estão acima desse valor. O percentil é útil para analisar a distribuição dos dados e identificar valores atípicos. Outros exemplos comuns de percentis são o quartil (divisão em quartos) e o decil (divisão em décimos). Cada percentil fornece uma visão específica sobre a distribuição dos dados.

Quartil

Os quartis são medidas estatísticas que dividem um conjunto de dados ordenados em quatro partes igualmente espaçadas. Existem três quartis: o primeiro quartil (Q1), o segundo quartil (Q2) e o terceiro quartil (Q3). O primeiro quartil, ou Q1, divide os dados em 25% inferiores e 75% superiores, representando o valor abaixo do qual 25% dos dados estão. O segundo quartil, ou Q2, é equivalente à mediana, dividindo os dados em 50% inferiores e 50% superiores. O terceiro quartil, ou Q3, divide os dados em 75% inferiores e 25% superiores. Os quartis são úteis para entender a dispersão e a distribuição dos dados, sendo amplamente utilizados em análises estatísticas e tomadas de decisão.

Medidas de Disperção

Amplitude

A amplitude é uma medida estatística simples que representa a diferença entre o maior e o menor valor em um conjunto de dados. É calculada subtraindo o menor valor do maior valor encontrado. A amplitude fornece uma ideia básica da variação dos dados, indicando a extensão do intervalo em que os valores estão distribuídos. No entanto, a amplitude pode ser sensível a valores extremos, sendo mais útil em conjuntos de dados que não apresentam valores discrepantes significativos.

Variância

A variância é uma medida estatística que indica a dispersão dos valores em relação à média. Ela representa a média dos quadrados das diferenças entre cada valor e a média do conjunto de dados. A variação é útil para entender o quão os valores individuais se afastam da média e como eles estão distribuídos em torno dela. Quanto maior a variância, maior é a dispersão dos dados. Por outro lado, uma variância baixa indica que os valores estão mais próximos da média. A variância é amplamente utilizada em estatística para análise e comparação de conjuntos de dados.

Desvio padrão

O desvio padrão é uma medida de dispersão que indica o quão distantes os valores estão da média em um conjunto de dados. Ele quantifica a variabilidade dos dados e é amplamente utilizado em análises estatísticas. O desvio padrão é calculado encontrando a diferença entre cada valor e a média, elevando essas diferenças ao quadrado, somando-as e dividindo pelo número de observações.

Em seguida, a raiz quadrada é aplicada ao resultado para obter o desvio padrão. Quanto maior o desvio padrão, maior a variabilidade dos dados. Ele é útil para comparar a dispersão de diferentes conjuntos de dados e identificar valores atípicos.
